# ynab-sync
> Sync YNAB with Nubank and Banco do Brasil

## Requirements

You need REDIS installed: https://redis.io/

## Installation

```sh
yarn install
```

## Usage example

Just run the command bellow to retrieve your NuBank information:

```sh
yarn start -u <NUBANK_LOGIN> -p 2018-01
```
